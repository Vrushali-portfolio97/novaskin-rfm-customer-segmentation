# NovaSkin RFM Customer Segmentation

This project analyzes customer behavior using RFM segmentation (Recency, Frequency, Monetary) for a skincare brand case study called **NovaSkin**.

## Project Overview
The goal of this project was to segment customers based on how recently they purchased, how often they purchased, and how much they spent. The analysis was performed in Python and visualized in Looker Studio.

## Tools Used
- Python
- Pandas
- Google Colab
- Looker Studio

## Key Results
- Total Customers: 300
- Total Revenue: €199,354.44
- Average Frequency: 5.4
- Average Recency: 114.34 days
- 4 customer segments identified: Champions, Loyal Customers, At Risk, Lost
- Champions contributed the highest share of revenue

## Dashboard Insights
- **Champions** generated the highest revenue contribution
- **Loyal Customers** showed strong repeat purchase behavior
- **At Risk** customers had lower recent engagement and need retention campaigns
- **Lost** customers had the highest recency and lowest purchase activity

## Files in This Repository
- `RFM_Customer_Segmentation.ipynb` — notebook used for RFM analysis
- `ecommerce_retail_data.csv` — input dataset
- `rfm_results.csv` — output customer segmentation results
- `RFM_Customer_Segmentation_Dashboard-2.pdf` — final dashboard export

## Business Value
This project shows how customer segmentation can support CRM and lifecycle marketing by helping brands create targeted campaigns for different customer groups.
